# sds_Spam_Email_Detection_System

This project detects spam emails, SMS messages, and URLs using Machine Learning.

## Features
- Email Scanner Detection
- Text Message Scanner Detection
- Email ID Checker Check
- URL ScannerDetection

## Installation

1. Clone the repository
git clone https://github.com/your-username/SDS_spam-email_detection_system.git

2. Go to project folder
cd SDS_spam-email-detection-system

3. Install requirements
pip install -r requirements.txt

4. Run the project
python app.py

## Requirements
- Python 3.10+
- Flask
- Scikit-learn

## Dataset
Use the SMS Spam Collection Dataset.
